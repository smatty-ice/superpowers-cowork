# Research Methodology

1. **Define questions.** Write down exactly what you need to learn before searching. Specific beats vague. See `references/research-process.md` for examples.
2. **Search 3+ query angles per question.** Direct, institutional, critical. Use WebSearch for each; follow up with WebFetch on the most promising results.
3. **Evaluate each source** for authority, recency, bias, and corroboration. Rubric in `references/source-evaluation.md`.
4. **Extract findings with attribution.** For every claim: source title, publisher, date, URL. Tag as fact / analysis / opinion.
5. **Hard stop:** Do not proceed to synthesis until you have at least 2 independent sources per key claim. If you cannot meet this bar, report the gap to the human before continuing.
6. **Synthesize.** Lead with consensus. Note disagreements. Identify gaps. Distinguish well-supported conclusions from tentative ones.
7. **Conflict rule:** When sources disagree, do not silently pick one — present both to the human and note which is better corroborated.
8. **Cite everything.** Every claim in the output traces back to a source (title, publisher, date, URL).

Key principle: **Triangulate.** One source is an anecdote. Three independent sources are evidence.
